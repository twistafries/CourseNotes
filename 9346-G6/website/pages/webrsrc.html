How is web content served?

Types of Web Resources

  • Static
    • Resources are served as it exists where it is located.
  • Dynamic
    • Content is generated on-the-fly, such as search engine results which are generated based on the query.

Client-Server Architecture

  • Network architecture consisting of servers which provide content or resources and clients that request and consume these resources.
